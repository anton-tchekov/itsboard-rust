# Rust on ITS-Board

Important:
The Program Version in ITS-Board-Uploader is already quite old!

To Upload most Up-To-Date Version:
Copy `analyzer/out.hex` to `itsboard-uploader/program/logic_analyzer.hex`

## Build and run

In `analyzer/` directory:

Build:

`make build`

Clean:

`make clean`

Flash:

`make flash`

ASM Dump:

`make dump`
